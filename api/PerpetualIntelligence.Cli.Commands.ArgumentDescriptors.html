    Item[String, Nullable<Boolean>, Nullable<Boolean>]

    Gets an ArgumentDescriptor instance with the specified id.

    Declaration
    public ArgumentDescriptor this[string idOrAlias, bool? alias = null, bool? ifNotThen = null ] { get; }
    Parameters
    Type Name Description
    String idOrAlias

    The argument id or its alias.
    Nullable<Boolean> alias

    true to find the argument by its alias, false to find by its identifier.
    Nullable<Boolean> ifNotThen

    true to find the argument by its id. If not found, then attempt to find it by its alias. false to find by id or alias but not both.
    Property Value
    Type Description
    ArgumentDescriptor

    ArgumentDescriptor instance if found.
    Remarks

    We recommend to use Item[String] to get an argument. Using alias will degrade the application's performance.
