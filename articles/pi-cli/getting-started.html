    Getting Started

    For test and evaluation

    You can use our demo license for quick onboarding, testing, and evaluating ready-to-use samples on GitHub.

    No account is needed.

    For community and education

    Our community license is free for education, research, and non-production use.

    Start by subscribing to the Community edition:

    • Subscribe on Microsoft AppStore
    • Subscribe on Microsoft Azure

    Once you activate the community subscription,

    1. Generate the license keys
    2. Download the ready-to-use CLI template
    3. Configure the CLI terminal with options
    4. Configure the template
    5. Add your command runners
    6. Test and run your terminal

    For commercial or production use

    Our commercial licenses and pricing model fits all. See available licenses for solo entrepreneurs, small-medium businesses, large enterprises, and service vendors.

    Start by subscribing to the Commercial edition:

    • Buy on Microsoft AppStore
    • Buy on Microsoft Azure

    Once you activate the commercial subscription,

    1. Generate the license keys
    2. Download the ready-to-use CLI template
