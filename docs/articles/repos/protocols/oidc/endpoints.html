    Endpoints

    The table below summarizes the OAuth 2.0 and OpenID Connect authorization server endpoint implementations. All OneImlx endpoints implements IAuthServerHandler<TContext, TResult>.

    Endpoint OneImlx Endpoint Id Protocol
    authorize urn:oneimlx:auth OAuth 2.0 Authorization Framework
    check_session or check_session_iframe urn:oneimlx:checksession OpenID Connect Session Management
    device_authorization urn:oneimlx:device OAuth 2.0 Device Authorization Grant
    discovery or .well-known urn:oneimlx:discovery OpenID Connect Discovery
    end_session urn:oneimlx:endsession OpenID Connect Session Management
    introspect urn:oneimlx:introspect OAuth 2.0 Token Introspection
    jwks urn:oneimlx:jwks OAuth 2.0 JSON Web Key (JWK)
    revoke urn:oneimlx:revoke OAuth 2.0 Token Revocation
    token urn:oneimlx:token OAuth 2.0 Authorization Framework
    user_info urn:oneimlx:userinfo OpenID Connect User Info
